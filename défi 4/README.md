## Installation

You need node.js and npm. You should probably install this globally. [Install Node](https://nodejs.org/)

```sh
git clone https://github.com/Esselka/BouletCanon.git
cd BouletCanon
npm install -g live-server
npx live-server
```

Enjoy !
